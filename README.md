						Welcome to MineSweeper
################################################################################


To play the game -> Open minesweeper.html in a web browser

Game controls : 
--Left click : To reveal a cell from the board
--Right click: To flag or unflag a cell 
  (Note: You cannot left click if cell is flagged)

Instructions :
--Every cell is either a bomb, holds a number or is blank.
--Number represents number of bombs in it's adjacent cells.
--Click on all cells that aren't bombs to win the game.
